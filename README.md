# Golden English Academy Website

Complete website for Golden English Academy - An English language school in São Paulo, Brazil.

## Features

### Main Pages
- **Homepage** - Landing page with course information, testimonials, and contact form
- **Level Test** - Interactive English proficiency test with 20 questions
- **Blog** - Educational articles about English learning
- **Contact** - Contact information and inquiry form
- **Login** - Student and admin login portal
- **Admin Dashboard** - Management system for courses, students, and scheduling

### Key Functionality
- Responsive design for all devices
- Interactive level testing system
- Course enrollment forms
- Contact management
- Admin panel with CRUD operations
- Modern UI with animations
- SEO optimized

## Technologies Used
- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
- Vite (build tool)
- Font Awesome icons

## Color Scheme
- Primary: Gold (#FFD700)

- Accent: Navy (#16213e)

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
```

### Preview Build
```bash
npm run preview
```
